vllm serve meta-llama/Llama-3-8B

# With TurboQuant — everything else identical
vllm serve meta-llama/Llama-3-8B --kv-cache-dtype turboquant
How it works: TurboQuant applies a random orthogonal rotation to each KV vector, then quantizes each coordinate to 2 bits using an optimal Lloyd-Max codebook for the rotation-induced Beta distribution. At read time, the query is rotated once instead of inverse-rotating every cache entry (query-rotation trick). The result is near-optimal distortion at 2 bits with zero calibration data.
